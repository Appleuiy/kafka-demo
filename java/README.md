# Java Kafka 示例

## 依赖配置

### Maven (pom.xml)
```xml
<dependencies>
    <dependency>
        <groupId>org.apache.kafka</groupId>
        <artifactId>kafka-clients</artifactId>
        <version>3.6.0</version>
    </dependency>
</dependencies>
```

### Gradle (build.gradle)
```gradle
dependencies {
    implementation 'org.apache.kafka:kafka-clients:3.6.0'
}
```
